#  AgriFlux

**AgriFlux** is an AI-powered agricultural intelligence platform that transforms raw farm data into real-time, actionable decisions.

It acts as a **virtual agronomist**, combining weather, soil, and crop data to guide farmers with precise recommendations.

---

## 🌍 Overview

AgriFlux reimagines farming as a **data-driven system** — helping farmers monitor, analyze, and optimize their fields through an intuitive dashboard.

From mapping farm plots to generating AI-based insights, AgriFlux delivers a complete **farm management experience**.

---

## 🚀 Core Features

### 🗺 Smart Farm Mapping

* Interactive plot selection using maps
* Polygon-based field creation
* Location-aware analysis

### 🌦 Weather Intelligence

* 7-day weather forecasts
* Evapotranspiration (ET₀) insights
* Soil moisture tracking

### 🧪 Soil Analysis

* Soil composition (pH, nitrogen, organic carbon, etc.)
* Hyper-local soil intelligence
* Nutrient-based recommendations

### 🤖 AI Agronomist

* Action-focused farming guidance
* Risk alerts (pests, diseases, weather)
* Structured, easy-to-understand reports

### 📊 Analytics Dashboard

* Crop progress tracking
* Yield and income projections
* Growth stage visualization

---

## 🧠 Tech Stack

* **Frontend:** Next.js (App Router), TypeScript
* **UI:** Tailwind CSS, Shadcn UI
* **State Management:** Zustand
* **Maps:** React-Leaflet
* **Charts:** Recharts
* **Animations:** Framer Motion
* **AI:** OpenAI API / compatible LLMs
* **Data APIs:** Open-Meteo, SoilGrids

---

## ⚡ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/roshanthm/AgriFlux
cd agriflux
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run Development Server

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env.local` file:

```env
OPENAI_API_KEY=your_api_key_here
```
